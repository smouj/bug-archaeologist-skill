Habilidad Arqueólogo de Errores para OpenClaw.

## Qué resuelve
Analiza errores y sus raíces históricas en el código para identificar patrones y prevenir su reincidencia.

## Características
- Activación automática al detectar tareas relevantes
- Resultados profesionales, listos para producción
- Enfoque primero en seguridad
- Soporte para rollback

## Uso
/bug-archaeologist

## Ejemplo
```
Analizar código en busca de problemas de análisis.
```

## Instalación
Esta habilidad está automáticamente disponible en OpenClaw.
